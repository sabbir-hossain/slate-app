
## Folder Structure

After creation, your project should look like this:

```
slate-app/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    components/
      slate.jsx
      toolbar.jsx
    App.css
    App.js
    App.test.js
    index.css
    index.js
    logo.svg
```


In the project directory, you can run:

### `npm i`
### `npm start`
